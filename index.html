1. INTRODUÇÃO

Este relatório descreve o projeto de um farol utilizando Arduino, com o objetivo de desenvolver habilidades em eletrônica e programação.

2. OBJETIVOS

O principal objetivo é criar um farol que simule as luzes de um sinalizador utilizando LEDs e um microcontrolador Arduino.

3. MATERIAIS UTILIZADOS

  • Arduino Uno
  • 3 LEDs (vermelho, amarelo, verde)
  • 3 resistores (220Ω)
  • Protoboard
  • Fios de conexão

4. DIAGRAMA DO CIRCUITO

Abaixo está o diagrama do circuito projetado no Tinkercad:

5. CÓDIGO FONTE

void setup() {
    pinMode(7, OUTPUT);
    pinMode(4, OUTPUT);
    pinMode(2, OUTPUT);
}

void loop() {
    digitalWrite(7, HIGH);
    delay(5000); 
    digitalWrite(7, LOW);

    digitalWrite(4, HIGH);
    delay(3000); 
    digitalWrite(4, LOW);

    digitalWrite(2, HIGH);
    delay(6000); 
    digitalWrite(2, LOW);
}

6. PROCEDIMENTOS

Os LEDs foram conectados aos pinos digitais do Arduino conforme o diagrama, e o código foi carregado na placa.

7. RESULTADOS

O farol funcionou conforme esperado, alternando as luzes de acordo com o tempo definido no código.

8. DISCUSSÃO

O funcionamento do circuito foi satisfatório, e os LEDs apresentaram um brilho adequado.

9. CONCLUSÃO

O projeto foi bem-sucedido, permitindo a prática de conceitos de eletrônica e programação.
